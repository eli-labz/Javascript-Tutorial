# JavaScript Course Interactive Tutorial

This workspace now includes a local interactive tutorial app that lets you browse the course lessons, read the task markdown, inspect the associated files, take quizzes on theory tasks, see next-lesson recommendations, and run runnable examples in a browser playground.

## Run the tutorial

```powershell
npm start
```

or

```powershell
npm run tutorial
```

Then open the URL shown in the terminal, usually:

```text
http://127.0.0.1:3000
```

## What it includes

- Lesson and task navigation for the full course
- Search across lesson and task titles
- Rendered markdown for `task.md`
- Quiz mode for theory tasks
- Automatic next-lesson recommendations
- File tabs for `task.js` and any extra support files
- Browser code playground for runnable examples
- Local progress tracking in your browser
- Previous / next navigation inside the course order

## Existing course tests

The original Jest test command still works:

```powershell
npm test
```
